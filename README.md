# kafka-examples
# Setup environment
- clone the repo
- ```vagrant up``` Brings the base virtual machine
- Take ssh to machine ```vagrant ssh```
- Switch to root ```sudo su - ```
- ```cd /root/cp-all-in-one/cp-all-in-one-community ```
- ```docker-compose up -d ```
# Confluent control center
- http://localhost:9021/
